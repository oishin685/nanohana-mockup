# 菜の花村 ウェブサイトリニューアル モックアップ

宮城県登米市の自然食品宅配店「菜の花村」のウェブサイトリニューアル用プレゼンモックアップです。

## プロジェクト概要

- **クライアント**: 菜の花村（自然食品宅配）
- **所在地**: 宮城県登米市中田町石森188-1
- **既存サイト**: http://www.nano87.com/
- **目的**: 既存の年配顧客を大切にしつつ、若い新規顧客を獲得するためのモダンなリニューアル

## ファイル構成

```
kinohanazono-mockup/
├── css/
│   └── style.css          # メインスタイルシート
├── js/
│   └── script.js          # JavaScript（メニュー、アニメーション、フォーム等）
├── images/                # 画像フォルダ（プレースホルダー使用中）
├── index.html             # トップページ
├── vegetables.html        # 野菜・卵カテゴリ
├── seasoning.html         # 調味料カテゴリ
├── processed.html         # 加工食品カテゴリ
├── rice.html              # お米カテゴリ
├── delivery.html          # 配達・注文について
├── about.html             # お店について
├── contact.html           # お問い合わせ・アクセス
└── README.md              # このファイル
```

## 技術仕様

- **HTML5** / **CSS3** / **Vanilla JavaScript**
- **レスポンシブデザイン**: モバイルファースト（480px / 768px / 1024px ブレークポイント）
- **フォント**: Google Fonts（Noto Sans JP、Noto Serif JP）
- **外部依存なし**: フレームワーク不使用、軽量な静的サイト

## デザイン特徴

- **カラーパレット**: ナチュラル系（グリーン・ベージュ・ホワイト基調）
- **アニメーション**: スクロールフェードイン（IntersectionObserver使用）
- **ナビゲーション**: PC版ドロップダウン / モバイル版ハンバーガーメニュー
- **プレースホルダー画像**: CSSグラデーション + 絵文字で表現（実画像は後日差し替え）
- **アクセシビリティ**: prefers-reduced-motion対応、ARIA属性使用
